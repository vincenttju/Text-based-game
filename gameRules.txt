GAME RULES :
- 7 important commands needed throughout the game:
    --> go
    --> take
    --> use
    --> examine
    --> view
    --> quit
- You can input  such as :
    --> go north
    --> take letter
    --> examine letter
    --> use key
    --> view inventory
- Input must be consist of 2 words and lowercase.
- You can abandon the game by inputting "quit game".
